Quick Guide
  1. Select quiz date and subject
  2. Paste questions in JSON format
  3. Click "Validate JSON" to verify
  4. Review question count and time
  5. Click "Create Quiz" to publish
JSON Template
[
  {
    "question": "What is the capital of India?",
    "options": [
      "Mumbai",
      "New Delhi",
      "Kolkata",
      "Chennai"
    ],
    "correctAnswer": 1,
    "explanation": "New Delhi is the capital of India",
    "timeAllocation": 60
  }
]
Pro Tips
  • correctAnswer is the index (0-3) of correct option
  • timeAllocation is in seconds (default: 60)
  • Always validate before creating quiz
  • Add detailed explanations for better learning
